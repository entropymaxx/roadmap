# 🚀 April Cyber-Learning Roadmap

This repository contains the source code for a dynamic, dark-mode study dashboard hosted on **GitHub Pages**. It is designed to track a rigorous 4-week learning protocol focusing on **Advanced Linux**, **Computer Networking (CS144)**, and **Web Hacking**.

---

## 🛠️ Purpose
The goal of this website is to provide a visually organized, interactive schedule for mastering low-level systems and network security. The roadmap is broken down into specific weekly milestones:

* **Week 1:** Internet Fundamentals, IP Core, and Addressing (IPv4/ARP).
* **Week 2:** Transport Layer reliability (TCP/UDP) and Congestion Control.
* **Week 3:** Application layers (HTTP/DNS), Routing essentials (BGP/Dijkstra), and the start of Web Hacking.
* **Week 4:** Network Security core (TLS/Crypto/DoS) and deep-dives into web vulnerabilities like CSRF and SSRF.

---

## 💻 Study Tracks
The plan executes three parallel tracks to ensure a well-rounded understanding of the stack:
1.  **Advanced Linux:** Scheduled classes focusing on system-level mastery.
2.  **CS144 Networking:** A trimmed selection of 50 essential lectures from the Stanford University networking curriculum.
3.  **Web Hacking:** Practical exploitation techniques based on Rana Khalil's methodology, covering SQLi, XSS, and Access Control.
